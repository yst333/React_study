※ reactstrap VS react-bootstrap 차이

1. reactstrap은 상태값(state)과 class 컴포넌트를사용하고,
   react-bootstrap은 function형 컴포넌트와 hooks를 사용한다고합니다.
   실제 사용에서는 거의 구분 없이 사용하고 있습니다.

2. Reactstrap npm install 문법(예시)
   npm install --save reactstrap react react-dom

3. React-Bootstrap npm install 문법(예시)
    npm install react-bootstrap bootstrap
